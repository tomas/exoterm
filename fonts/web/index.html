pixelfonts

A browser-based bitmap & pixel font previewer. Browse, filter, preview, and export fonts — no install required.

Supported formats

.bdf .pcf .pcf.gz .ttf .otf .fon .fnt

Drop any supported font onto the page to load it. TTF/OTF fonts are automatically analyzed for pixel-perfect sizing — non-bitmap fonts are rejected.

Fonts can be exported as BDF or PCF, with optional embolden and scale transformations applied.

Font library sourced from Tecate/bitmap-fonts and other open sources.
